# Tags publicadas

Índice legível das tags que correspondem a releases com build/deploy no GitHub Actions. O Git (refs no remoto) continua a ser a fonte de verdade; este ficheiro é convenção de equipa.

**Atualiza `TAG.md` antes de `git push origin vX.Y.Z`**, para refletir a tag que vais publicar (e alinhar com `CHANGELOG.md`).
